# 江南案互動式歷史紀錄網站 — 實作計畫

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 建立一個 scrollytelling 風格的互動式歷史紀錄網站，以「解密檔案」形式呈現 1984 年江南案的歷史全貌。

**Architecture:** React SPA 搭配 PixiJS pixel art 場景。資料層從版權書籍提取為結構化 JSON/MD（書籍不入版控），敘事層以章節為單位組織，視覺層結合 TailwindCSS 報紙/檔案風主題與 PixiJS 像素場景。

**Tech Stack:** TypeScript, React 18, Vite, TailwindCSS v4, PixiJS v8, GSAP ScrollTrigger

**資料來源特殊說明：** 五本書（龍之火上下、董桂森、吳敦回憶錄、忠與過）為版權書籍，`data/` 目錄須加入 `.gitignore`。Phase 1 完成後，所有後續開發僅依賴 `docs/extracted/` 的結構化資料。
