# Pixel Art 場景規格

> 所有場景使用 PixiJS v8 渲染，封裝為 React 元件。風格參考 *Papers, Please*，但色調偏暖，配合報紙/解密檔案風格。

---

## 共用規範

| 項目 | 規格 |
|------|------|
| 基礎解析度 | 320 × 180 px |
| 縮放方式 | Integer scale up，保持像素銳利（`antialias: false`） |
| 色彩深度 | 每場景限 16-24 色（含透明） |
| 動畫輸入 | `progress` prop（0.0 - 1.0），由 ScrollTrigger 驅動 |
| 手機降級 | 靜態 PNG（從 progress=0.5 截取）或簡化 CSS 動畫 |
| 資源管理 | 離開 viewport 後釋放 WebGL 資源、銷毀 PixiJS Application |
| 載入策略 | IntersectionObserver lazy load，進入 viewport 前 200px 開始初始化 |
| Sprite 管理 | 每場景一張 sprite sheet（PNG + JSON atlas） |

## 場景 1：`garage-shot`

**所屬章節：** 序章 — 車庫裡的槍聲
**情緒基調：** 寂靜中的突然暴力。冷色黎明 → 暖色槍焰 → 暗紅擴散。

### 色調

| 用途 | 色碼 | 說明 |
|------|------|------|
| 天空 | `#2d3a4a` | 破曉前的深藍灰 |
| 車庫牆壁 | `#5a5248` | 水泥灰褐 |
| 地面 | `#4a4540` | 車庫地板深灰 |
| 人物剪影 | `#1a1816` | 近乎黑色 |
| 槍焰 | `#e8a832` → `#c23b22` | 黃→紅，2 幀閃爍 |
| 血跡 | `#8b2020` | 暗紅，低飽和 |
| 光源（車庫門外） | `#c9b99a` | 清晨微光 |

### 元素清單

| 元素 | 尺寸 (px) | 說明 |
|------|----------|------|
| 車庫外景 | 320×60 | 天空 + 住宅外牆上半部 |
| 車庫門 | 120×80 | 半開狀態，光線從外透入 |
| 樓梯 | 40×90 | 畫面左側，通往二樓 |
| 柱子 | 12×70 | 吳敦躲藏處 |
| 洗衣機 | 24×28 | 畫面右側，董桂森躲藏處 |
| 吳敦剪影 | 16×32 | 蹲姿，在柱子後方 |
| 董桂森剪影 | 16×32 | 蹲姿，在洗衣機旁 |
| 劉宜良 | 16×36 | 從樓梯走下 |
| 槍焰效果 | 8×8 | 2 幀動畫 sprite |
| 血跡擴散 | 32×16 | 4 幀漸擴動畫 |
| 自行車 ×2 | 20×16 | 畫面外側，撤離用 |

### Progress 動畫行為

| Progress 範圍 | 動畫 |
|--------------|------|
| 0.00 - 0.15 | 靜態車庫場景。微光從門外透入，輕微閃爍。吳敦、董桂森的剪影隱約可見。 |
| 0.15 - 0.25 | 劉宜良的身影從樓梯上方緩慢走下（逐格動畫，每步 3 幀）。 |
| 0.25 - 0.30 | 劉宜良停頓，轉頭看向暗處（1 秒靜止）。 |
| 0.30 - 0.35 | **第一槍**：吳敦位置閃出槍焰（2 幀）。全場景白閃 1 幀。劉宜良向後倒。 |
| 0.35 - 0.45 | **第二、三槍**：董桂森位置閃出槍焰（每次 2 幀）。血跡從劉宜良身下擴散（4 幀）。 |
| 0.45 - 0.60 | 兩道剪影從車庫門離開。車庫恢復寂靜。光影不變。 |
| 0.60 - 0.80 | 畫面緩慢 zoom out，露出住宅全貌和寧靜的郊區街道。對比。 |
| 0.80 - 1.00 | 畫面漸暗（fade to near-black），只剩車庫門口的一縷晨光。 |

## 場景 2：`spy-network`

**所屬章節：** 第一章 — 龍的爪牙
**情緒基調：** 冷戰氛圍。技術感、監控感。冷色調為主，偶有紅色警示。

|------|------|------|

| 大陸板塊 | `#3d4a3d` | 中等軍綠 |
| 台灣 | `#b8943e` | 金色高亮 |
| 連線（活躍） | `#4a6fa5` | 藍色脈衝 |
| 連線（歷史） | `#5a5248` | 灰褐 |
| 節點（領事館） | `#c23b22` | 紅色圓點 |
| 節點（情報站） | `#e8a832` | 金色方點 |
| 文字標籤 | `#c9b99a` | 淡紙色 |

### 元素清單

| 元素 | 尺寸 (px) | 說明 |
|------|----------|------|

| 台灣島 | 8×16 | 金色高亮，中心有脈衝效果 |
| 美國輪廓 | 80×40 | 標示 15 個領事館位置 |
| 領事館節點 ×15 | 3×3 | 紅色圓點 |

| 標籤文字 | 像素字型 | 城市名（英文縮寫） |

### Progress 動畫行為

| Progress 範圍 | 動畫 |
|--------------|------|
| 0.00 - 0.20 | 空白地圖，僅顯示大陸板塊輪廓。台灣從黑暗中亮起金色。 |

| 0.40 - 0.60 | 第二波連線延伸至美國。15 個領事館節點逐一亮起。資訊脈衝開始在連線上流動。 |
| 0.60 - 0.80 | 所有連線活躍，脈衝加速。數字計數器顯示「特工人數：50,000+」。 |
| 0.80 - 1.00 | 美國位置出現一個紅色警告標記，文字閃爍「HOSTILE」。全圖微微震動。 |

## 場景 3：`typewriter-study`

**所屬章節：** 第二章 — 江南其人
**情緒基調：** 孤獨、執著、書卷氣。暖色燈光下的寫作者。

### 色調

| 用途 | 色碼 | 說明 |
|------|------|------|
| 房間背景 | `#3d3833` | 深褐 |
| 書桌 | `#6b5d4d` | 木質褐 |
| 打字機 | `#2d2a26` | 深黑 |
| 紙張 | `#f5f0e8` | 紙張白 |
| 檯燈光暈 | `#e8c878` | 暖黃 |

| 窗外 | `#2d3a4a` | 夜色深藍 |
| 曇花 | `#f0e8d8` | 淡白 |

### 元素清單

| 元素 | 尺寸 (px) | 說明 |
|------|----------|------|

### Progress 動畫行為

| Progress 範圍 | 動畫 |
|--------------|------|
| 0.00 - 0.20 | 黑暗房間。檯燈亮起（從暗到明，3 幀）。光暈逐漸擴大，照亮桌面和打字機。 |

| 0.40 - 0.55 | 書架逐步被燈光照亮。書脊上的文字若隱若現。手稿堆漸高（每隔幾幀加一張紙）。 |

| 0.70 - 0.85 | 曇花在盆栽中緩慢綻放（4 幀動畫）——敘事中「曇花一現」的象徵。灰色封面書出現在桌上。 |
| 0.85 - 1.00 | 燈光漸暗。只剩窗外晨光和曇花的輪廓。暗示命運逼近。 |

## 場景 4：`bamboo-meeting`

**所屬章節：** 第三章 — 竹林裡的兄弟
**情緒基調：** 暗流湧動的社交場合。表面觥籌交錯，暗處是權力的計算。

### 色調

| 用途 | 色碼 | 說明 |
|------|------|------|
| 房間背景 | `#3a2a20` | 深暗紅褐（中式餐廳） |
| 圓桌 | `#6b4d3d` | 紅木色 |
| 桌布 | `#8b3030` | 暗紅 |
| 酒杯/碗盤 | `#c9b99a` | 陶瓷淡色 |
| 人物 A（汪希苓） | `#3d5040` | 軍綠基調 |
| 人物 B（陳啟禮） | `#2d2a26` | 深色西裝 |
| 人物 C-F（其他賓客） | 各異 | 低飽和色 |
| 燈光 | `#e8b050` | 暖黃吊燈 |
| 陰影 | `#1a1412` | 深暗 |

### 元素清單

| 元素 | 尺寸 (px) | 說明 |
|------|----------|------|
| 房間背景 | 320×180 | 中式餐廳內景 |
| 圓桌 | 100×50 | 俯視角度略帶透視 |
| 吊燈 | 24×16 | 畫面上方中央 |
| 人物剪影 ×6-8 | 16×28 | 圍坐圓桌，各有特色 |
| 酒杯 ×8 | 4×6 | 桌上 |
| 菜盤 ×4 | 12×8 | 桌上 |
| 暗處的眼睛 | 4×2 | 汪希苓的特寫，在某個 progress 階段閃現 |

### Progress 動畫行為

| Progress 範圍 | 動畫 |
|--------------|------|
| 0.00 - 0.15 | 空蕩的圓桌，吊燈搖晃（2 幀）。酒杯反光。 |
| 0.15 - 0.35 | 人物一一入座（每人 2 幀入場動畫）。先是普通賓客，然後陳啟禮（深色），最後汪希苓（軍綠）。 |
| 0.35 - 0.55 | 觥籌交錯動畫：舉杯、放下、夾菜（循環）。表面的歡聚。 |
| 0.55 - 0.70 | 燈光稍暗。汪希苓的方向出現「觀察」的眼神特寫（zoom in 到臉部，像素放大）。暗示暗流。 |
| 0.70 - 0.85 | 其他賓客逐漸暗化為剪影。只有汪希苓和陳啟禮保持亮度。兩人之間出現一條隱約的紅色連線。 |
| 0.85 - 1.00 | 場景整體暗化，只剩桌上一杯酒的反光和那條紅色連線。預示陰謀。 |

## 場景 6：`prison-fade`

**所屬章節：** 第六章 — 審判與餘波
**情緒基調：** 囚禁、孤寂、漸漸消逝。從有到無。

### 色調

| 用途 | 色碼 | 說明 |
|------|------|------|
| 牢房牆壁 | `#4a4a4a` | 冷灰 |
| 鐵窗 | `#2d2d2d` | 深灰 |
| 地面 | `#3a3a3a` | 灰色 |
| 窗外光線 | `#6a7a8a` | 冷藍灰 |
| 人物（董桂森） | `#5a5248` | 灰褐 |
| 信紙 | `#f5f0e8` | 紙白 |
| 血跡 | `#6b2020` | 暗紅，極低飽和 |

### 元素清單

| 元素 | 尺寸 (px) | 說明 |
|------|----------|------|
| 牢房背景 | 320×180 | 灰色石牆 |
| 鐵窗 | 40×50 | 畫面左上，有柵欄 |
| 床鋪 | 48×20 | 畫面右下 |
| 小桌 | 24×16 | 床鋪旁 |
| 信紙 ×3 | 8×10 | 桌上或地上散落 |
| 董桂森 | 16×32 | 坐在床邊或站立 |
| 窗外光束 | 60×120 | 半透明斜光 |

### Progress 動畫行為

| Progress 範圍 | 動畫 |
|--------------|------|
| 0.00 - 0.20 | 牢房場景建立。窗外有光束斜射入。董桂森坐在床邊，手持信紙。靜態。 |
| 0.20 - 0.40 | 信紙從桌上一張張飄落至地面（2 幀動畫×3）。時間流逝——代表他寫給母親和家人的信。 |
| 0.40 - 0.55 | 窗外光線從冷藍灰變為更暗的色調。日夜交替（2-3 個循環），暗示歲月流逝。 |
| 0.55 - 0.70 | 董桂森站起，走向牢房門方向。忽然畫面一閃——暗紅色出現在胸口位置。他的手按住胸口。 |
| 0.70 - 0.85 | 董桂森緩慢倒下（4 幀）。信紙散落。窗外光線漸暗。 |
| 0.85 - 1.00 | 整個場景逐漸消融——不是簡單的 fade out，而是像素逐個消失（dither 效果），從邊緣向中心。最後消失的是窗外那道光。全黑。 |
