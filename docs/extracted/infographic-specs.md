# 資訊圖表規格

> 所有資訊圖表使用 SVG + CSS 實作（不用 PixiJS），以確保無障礙、SEO 友善及 RWD 相容性。互動部分使用 React 狀態管理。

---

## 共用規範

| 項目 | 規格 |
|------|------|
| 渲染技術 | SVG（靜態結構） + CSS 動畫 + React 狀態管理（互動） |
| 色彩系統 | 遵循 CLAUDE.md 定義的色彩（paper、ink、stamp-red、link-blue、evidence-gold、neutral） |
| 字型 | 中文：serif（與正文一致）；英文數據：JetBrains Mono |
| 無障礙 | 所有圖表附 `aria-label`；關鍵資訊有文字替代方案；支援鍵盤導航 |
| 動畫 | 尊重 `prefers-reduced-motion`；關閉動畫時顯示完整靜態版本 |
| 資料來源 | 從 `src/data/` 引入 JSON，不硬編碼 |

```
                    蔣經國（總統）
                        │
            ┌───────────┼───────────┐
            │           │           │
        國家安全局    國防部      行政院
        (汪敬煦)       │        調查局
            │      ┌───┴───┐
            │   情報局    政戰部

            │ (陳虎門) (胡儀敏)

- 每個節點為「檔案卡片」風格（paper-100 背景、ink-700 邊框）

| 平板 | 簡化為兩層（頂層 + 情報局內部），tap 互動 |

**所屬章節：** 第四章 — 鋤奸

- `docs/extracted/contradictions.json`（`mission-cancellation`、`sanction-order-timing`）

### 視覺設計

```
台灣時間 (GMT+8)          美國太平洋時間 (GMT-7)
────────────────          ──────────────────────
10/15 上午                10/14 下午
  │                         │
  ├─ 汪希苓告訴陳虎門       │
  │  「不要勉強，叫他       │
  │   們回來好了」           │
  │                         │
  │    ←── 時差 15 小時 ──→  │
  │                         │
10/16 上午                10/15 上午
  │                         │
  ├─ 陳虎門接到電話          ├─ 09:00 劉宜良走下樓梯
  │  「買賣已成，             ├─ 09:58 法醫確認死亡

- 左軸：台灣時間線（`link-blue` 色）

| 矛盾標記 | 點擊紅色框展開四方對照（ContradictionBox） |
| 時差視覺化 | 動態虛線連接兩軸同一時刻，強調時差的致命性 |

### RWD 降級

| 裝置 | 呈現方式 |
|------|---------|
| 桌面 | 雙軸並列，hover 互動 |
| 平板 | 雙軸並列，tap 互動 |

## 圖表 3：`contradiction-table`

**所屬章節：** 第四章（主要）、第五、六章穿插
**類型：** 固定左欄四方對照表

### 資料來源

- `docs/extracted/contradictions.json`（全部 24 條矛盾）

### 視覺設計

```
┌──────────────┬──────────────┬──────────────┬──────────────┬──────────────┐
│   議題        │  汪希苓       │  吳敦        │  董桂森/梁東屏│  Kaplan      │
│  (固定左欄)   │  (忠與過)     │  (回憶錄)    │  (槍手書)     │  (龍之火)    │
├──────────────┼──────────────┼──────────────┼──────────────┼──────────────┤
│ 教訓 vs 制裁  │ 法庭：教訓    │ 自始至終：   │ 陳虎門承認   │ 傾向認定     │
│              │ 訪談：制裁    │ 制裁         │ 教訓是假的   │ 殺人命令     │
│              │              │              │              │              │
│ [stamp-red   │              │              │              │              │
│  矛盾標記]   │              │              │              │              │
├──────────────┼──────────────┼──────────────┼──────────────┼──────────────┤
│ ...          │ ...          │ ...          │ ...          │ ...          │
└──────────────┴──────────────┴──────────────┴──────────────┴──────────────┘
```

- 左欄（議題）固定不捲動，背景 `paper-200`
- 四方欄位：各用微妙不同的背景色區分
  - 汪希苓：`#f0ede6`（偏暖灰）
  - 吳敦：`#edf0e6`（偏綠灰）
  - 董桂森：`#f0e6e6`（偏紅灰）
  - Kaplan：`#e6edf0`（偏藍灰）
- 矛盾處以 `stamp-red` 斜線底紋標記
- 有一致結論的行用 `evidence-gold` 底色
- significance 等級用圖示標記：`critical` = 紅色驚嘆號、`major` = 橙色三角、`minor` = 灰色圓點

### 互動方式

| 互動 | 行為 |
|------|------|
| Hover 行 | 整行高亮，方便橫向對比 |
| 點擊議題 | 展開完整引述（含來源頁碼） |
| 篩選 | 可按 significance 篩選（critical / major / minor） |
| 章節篩選 | 可按所屬章節篩選（只看第四章的矛盾等） |
| 排序 | 按 significance 或章節排序 |

### RWD 降級

| 裝置 | 呈現方式 |
|------|---------|
| 桌面 | 完整四欄表格，固定左欄，橫向捲動 |
| 平板 | 三欄可見 + 橫向捲動 |
| 手機 | 卡片式。每個議題一張卡，tab 切換四方說法 |

## 圖表 7：`character-map`

**所屬：** 全域元素（浮動面板，任何章節可開啟）
**類型：** 中心放射狀人物關係圖

### 資料來源

- `docs/extracted/characters.json`（全部 94 人，但僅顯示核心 20-25 人）

### 視覺設計

```
                    國際
                   (灰色)
                  FBI·CIA
                    │
        家人 ───── 劉宜良 ───── 情治機關
       (暖色)     (中央)        (冷色/軍綠)
      崔蓉芝                   汪希苓
                                陳虎門
                                胡儀敏
                    │
        竹聯幫 ─────┘───── 蔣家
       (深色)              (金色)
      陳啟禮              蔣經國
      吳敦                蔣孝武
      董桂森
      張安樂

        調查者
       (藍色)
      劉善謙
      約翰·華倫
```

- 劉宜良在正中央，金色圓框，最大節點（24px 半徑）
- 六組人物以不同色系分區（參照 CLAUDE.md）：
  - 家人：`#c9886a`（暖褐）
  - 竹聯幫：`#3d3833`（深色）+ `#f5f0e8` 文字
  - 情治機關：`#4a5a48`（軍綠）
  - 蔣家：`#b8943e`（金色）
  - 調查者：`#4a6fa5`（藍色）
  - 國際：`#8b7d6b`（灰色）
- 節點大小依重要性分級：核心人物（16px）、次要人物（10px）、背景人物（6px）
- 關係連線：
  - 實線 = 直接關係（上下級、親屬）
  - 虛線 = 間接關係（傳聞、推測）
  - 紅色線 = 衝突/對抗
  - 雙向箭頭 = 互動頻繁

### 互動方式

| 互動 | 行為 |
|------|------|
| Hover 節點 | 高亮該人物的所有關係線；其他人物暗化 |
| 點擊節點 | 展開 DossierCard（檔案卡片）：照片（像素風）、姓名、別名、角色、生卒、與案件關係、出現章節 |
| 章節篩選 | 自動根據當前閱讀章節高亮相關人物 |
| 群組篩選 | 可按群組開關顯示/隱藏 |
| 搜尋 | 支援人名搜尋（中英文） |
| 展開/收合 | 面板可最小化為右下角小圖標 |

### RWD 降級

| 裝置 | 呈現方式 |
|------|---------|
| 桌面 | SVG 放射狀圖，浮動面板佔 40% 寬度 |
| 平板 | 同桌面但佔 60% 寬度 |
| 手機 | 全螢幕模態（modal），可捏合縮放（pinch zoom），或改為分組列表模式 |
